# XRAY Delivery Playbook

1. Intake Project/Squad/Epic and classify target artifact.
2. Build Requirement Inventory from approved source documents.
3. Create Test Plan when scope/phase needs to be explicit.
4. Generate API/UI/UAT Test Design as Test Conditions.
5. Generate Test Cases with contract-compatible field contract.
6. Build Test Set by execution purpose.
7. Generate Test Execution Pack for a run instance.
8. Export testcase/execution TSV when operational handoff is needed.
9. Run coverage audit and output contract review.
