# XRAY-style Artifact Mapping

| Artifact | BIDV file | Rule |
|---|---|---|
| Requirement | RequirementInventory.md | Source being verified |
| Test Plan | TestPlan.md | Scope and strategy |
| Test Design/Test Condition | TestDesign.md | Coverage intent |
| Test Case/Test | TestCase.md | Reusable test logic |
| Test Set | TestSet.md | Group of tests for a purpose |
| Test Execution | TestExecution.md | Specific run instance |
| Coverage Matrix | CoverageMatrix.md | Gap and traceability audit |
| Output Review | OutputReview.md | Handoff decision |
