# Testcase Excel/TSV Column Contract

Các cột bắt buộc cho testcase export:

| Column | Required | Purpose |
|---|---:|---|
| Test Suite Name | Yes | Nhóm suite/import |
| Details | No | Mô tả suite hoặc module |
| Testcase LV1 | Yes | Cấp nghiệp vụ/module |
| Testcase LV2 | Yes | Cấp chức năng/màn hình/API |
| Testcase LV3 | No | Cấp scenario nhóm |
| External ID | Yes | ID testcase ổn định |
| Name | Yes | Tên testcase |
| Summary | Yes | Tóm tắt mục tiêu kiểm thử |
| PreConditions | Yes | Điều kiện trước khi chạy |
| Status | Yes | Draft/Ready/Blocked |
| ExecutionType | Yes | Manual/Automated Candidate |
| Importance | Yes | High/Medium/Low |
| Keywords | No | Tags |
| Number of Attachments | No | Số evidence/attachment dự kiến |
| Step | Yes | Bước thực hiện |
| Expected Result | Yes | Kết quả mong đợi |
| Actual Result | No | Kết quả thực tế khi execute |
| StepExec Type | Yes | Manual/Automated |
| Spec Title | Yes | Tên spec/source |
| Document ID | Yes | Requirement/source document ID |
| Estimated exec. Duration | No | Thời gian dự kiến |
| Requirement ID | Yes | Trace về requirement |
| Test Condition ID | Yes | Trace về test condition |
| Evidence Required | Yes | Evidence cần thu thập |
| Open Question / Dependency | No | Câu hỏi/phụ thuộc |
| Execution Eligibility | Yes | Ready/Blocked/Need Review |
